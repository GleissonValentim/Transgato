Transgato App

Aplicativo Android desenvolvido para auxiliar na comunicação interna da empresa Transgato.

- Visão Geral

O Transgato é um aplicativo desenvolvido em *Kotlin* utilizando o ecossistema padrão do Android.  
Seu objetivo é facilitar a comunicação e melhorar a organização dos processos internos da empresa.

- Tecnologias Utilizadas

- Kotlin
- Android Studio
